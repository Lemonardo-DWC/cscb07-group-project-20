## Overview
This project is an Android app designed for use by healthcare providers, parents, and children.

This app is kid-friendly and supports managing and monitoring asthma-related activities in real time, including symptom tracking, medication logging, and exporting shareable reports.

## Features

### General Features:
- Secure sign-in with recovery options.
- Role selection (parent, children, healthcare provider).
- Entry author records for logs.
- History browser.

### For Children:
- Gamified UI to encourage app usage.
- Daily check-ins.
- Ability to log symptoms and medicine usage.
- Easy access to tips on technique and simple guidance.
- Each child will only see their own data, allowing for quick and clear navigation and access.
- Emergency communication protocol:
    - Automatic escalation.
    - Parental alert.


### For Parents:
- Ability to manage one or multiple children profiles.
- Access to a dashboard displaying key information.
- Receives important alerts in app and via notification.
- Information sharing, including report generation (PDF/CSV).
- Control over information sharing with healthcare provider(s).
- Adherence calculation based on controller logs compared to planned schedule.
    - Planned schedules are configurable by parents.

### For Healthcare Providers:
- Read-only access to reports from parents via a one-time invite code/link.
